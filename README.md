# Manipulación y Conversión de Archivos con Python y Pandas

## Descripción del Proyecto

Este proyecto demuestra el manejo integral de distintos tipos de archivos utilizando Python, principalmente con la librería Pandas.  

El objetivo es evidenciar competencias en lectura, escritura, transformación, conversión y normalización de datos provenientes de múltiples formatos, así como operaciones básicas sobre bases de datos.

Se trabajó con archivos CSV, Excel, Google Sheets, JSON, HTML, XML y bases de datos relacionales.

---

## Tecnologías Utilizadas

- Python 3
- Pandas
- NumPy
- OpenPyXL
- SQLite / Base de datos relacional
- Google Sheets API
- JSON
- XML
- HTML

## Funcionalidades Implementadas

### 1. Archivos CSV
- Lectura de archivos CSV.
- Escritura y exportación de nuevos archivos CSV.
- Limpieza y transformación de datos.
- Conversión de CSV a otros formatos.

### 2. Archivos Excel
- Lectura de archivos `.xlsx`.
- Escritura de múltiples hojas.
- Conversión de Excel a CSV.
- Transformación y procesamiento con Pandas.

### 3. Google Sheets
- Lectura de datos desde Google Sheets.
- Conversión de Google Sheets a DataFrame.
- Exportación de datos hacia nuevas hojas.

### 4. Archivos JSON
- Lectura de archivos JSON.
- Normalización de estructuras anidadas.
- Conversión de JSON a DataFrame.
- Exportación a CSV o Excel.

### 5. Archivos HTML y XML
- Extracción de tablas desde HTML.
- Parseo de archivos XML.
- Conversión a estructuras tabulares.
- Transformación y limpieza de datos.

### 6. Base de Datos
- Conexión a base de datos relacional.
- Lectura de tablas.
- Inserción de registros.
- Actualización de datos.
- Eliminación de registros.
- Consultas SQL ejecutadas desde Python.

## Operaciones Realizadas con Pandas

- Limpieza de datos.
- Manejo de valores nulos.
- Filtrado y selección de columnas.
- Agrupaciones (`groupby`).
- Transformaciones y agregaciones.
- Conversión entre distintos formatos de archivo.
